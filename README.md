# Challenge-Data-Scientist

Este repositorio contiene la resolución del desafío Data Scientist de Latam Airlines. Este archivo fue ejecutado en Google Colab pero se añaden todas las dependencias utilizadas en el archivo requirements.txt.

El notebook solution.ipynb contiene las respuestas a todas las preguntas planteadas, así como el análisis exploratorio y entrenamiento del modelo de predicción de retraso en vuelos.

Nota: La sección asociada a la optimización de hiperparámetros con GridSearch tarda más de una hora en su ejecución. Por lo tanto, para seguir con el análisis de explicabilidad e importancia de features, se entregan los hiperparámetros obtenidos mediante este proceso para que no tenga que ser ejecutado nuevamente.
